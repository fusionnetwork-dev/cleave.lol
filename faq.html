faq

frequently asked questions

general

+ is cleave free to use?
yes, completely free. no premium features, no paywalls, no bullshit. all features are available to everyone.
+ why should i use cleave over other bots?
cleave is built different. we dont have stupid premium features or paywall basic commands. everything is free, fast, and actually works. plus we have unique features like the vouch system, marriage, and a full economy.
+ what programming language is cleave built with?
python using discord.py. its fast, reliable, and gets constant updates.
+ how often is cleave updated?
we push updates weekly with new features and bug fixes. major updates come out monthly. check the status page for recent changes.

setup & usage

+ what permissions does cleave need?
for full functionality, give cleave administrator permission. if you want specific permissions, check the docs page for a detailed list. at minimum, cleave needs manage messages, ban members, kick members, and moderate members for mod commands to work.
+ how do i set up the bot?
after inviting cleave, run /help to see all commands. use /setlog, /setwelcome, /setautorole, etc to configure the bot. full setup guide is in the docs.
+ why isnt the bot responding?
make sure cleave has permissions in the channel. check that the bot is online (green status). if its still not working, try re-inviting the bot or join our discord for support.
+ can i customize the bot's prefix?
cleave uses slash commands (/) which cant be changed. this is discord's new standard and makes commands easier to use.
+ why cant i use certain commands?
you need specific permissions to use mod commands. check the docs for permission requirements. if youre admin and still cant use commands, make sure cleave's role is high enough in the role hierarchy.

features

+ how does the economy work?
everyone starts with $1000. earn money with /work and /crime. rob other users with /rob. gamble with /gamble. keep money safe in your bank with /deposit. check leaderboard with /leaderboard.
+ what is the vouch system?
vouch for trusted users with /vouch. this builds their reputation. useful for trading servers to see who's legit. check reps with /rep.
+ how does the starboard work?
set it up with /setstarboard #channel 3. when a message gets 3 ⭐ reactions, it appears in the starboard channel. great for highlighting funny or important messages.
+ what are counting channels?
setup with /counting #channel. users count sequentially (1, 2, 3...). cant count twice in a row. wrong number resets the count. simple but addicting.

troubleshooting

+ cleave is offline, what do i do?
check the status page. if its a planned outage, well show it there. for unexpected downtime, check our discord server for updates. we usually fix issues within minutes.
+ commands arent showing up
make sure cleave has "use slash commands" permission. try kicking and re-inviting the bot. sometimes discord needs a few minutes to sync commands.
+ the bot is lagging
this is rare. check our status page to see if theres an issue. if everything looks good on our end, the problem might be discord's api. wait a few minutes and try again.
+ found a bug, how do i report it?
join our discord server and open a ticket. include what you were doing, what happened, and screenshots if possible. we fix bugs fast.
+ a command gave an error
screenshot the error and send it in our discord. most errors are permission issues or discord rate limits. well help you figure it out.

privacy & data

+ what data does cleave store?
only necessary data: server configs (log channels, welcome channels, etc), economy balances, warnings, and vouch data. we dont collect personal info, dms, or message content.
+ can i delete my data?
yes. join our discord and request data deletion. we'll wipe all your data within 24 hours. removing the bot from your server also deletes server-specific data.
+ do you sell user data?
hell no. we dont sell, share, or give away any data. your info stays private. we're not scumbags.
+ is my economy data safe?
yes. all data is backed up regularly. if something breaks, we can restore it. weve never lost user data.
